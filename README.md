# Russ's Games

A collection of JavaScript games built with the help of code tutoring from ClaudeAI.

## About

I'm a computer science enthusiast and lifelong coder. My gaming history started on Commodore 64 BASIC in Italy during the 1980s, where I founded NICE (Northern Italian Computer Enthusiasts) and wrote the original *Escape from Mount Corna* in C64 BASIC. Decades later, I've revisited these classics and built new games using modern web technologies.

This repository is a mix of nostalgia, learning, and sharing—mostly for friends and family, but open to anyone curious about web-based games and retro computing. :)

## Technical Notes

- Built with vanilla HTML5, CSS, and JavaScript
- No external dependencies—everything runs client-side in your browser
- Tested on modern browsers (Chrome, Firefox, Safari, Edge)
- Game saves stored in browser localStorage where applicable

## Why These Games?

I've spent decades working in electronic security and life safety systems. Gaming and coding are where I keep my mind sharp and hands-on. These projects are part of learning, experimenting, and proving that old ideas can work in new contexts.

If you're learning to code, feel free to fork this repo and tinker. That's what it's here for.
